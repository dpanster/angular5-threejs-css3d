A Three.js - CSS3d showcase with terrain and physics...

This canvas contains two scenes from THREE.js and two renderer elements, a WebGLRenderer and a CSS3DRenderer.
The focus is the integration and handling on the CSS 3d in the scene. Features for this showcase are clickable, highlight, moveable css elements. One drop-down menu is integrated.
Additionally the WebGL scene contains a random generated terrain, light, shadow, camera handling with mouse, physics with collision detection, tween, object scaling, rotation around a (pivot-)point and a simple particle system. Play with the clickable elements and see whats happen.

A list of used libraries / packages for this angular controller example:
  • Three.js R91 (V0.91.0) / NPM: npm i three / main 3D part
  • CSS3dRenderer (V1.0.1) / NPM: npm i three-css3drenderer / CSS part
  • Trackball controls (V0.0.7) / NPM: npm i three-trackballcontrols / Camera control with mouse
  • Tween (V16.6.0) / NPM: npm i @tweenjs/tween.js / Soft moving objects
  • Cannon (V0.6.1) / NPM: npm i cannon / Physics
  • Perlin Noise (V0.0.1) / NPM: npm i perlin-noise / Noise for terrain
  • Chroma (V1.3.7) / NPM: npm i chroma-js / Color scaling for heightfield
