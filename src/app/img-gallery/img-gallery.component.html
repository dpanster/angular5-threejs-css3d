A Three.js - CSS3d showcase for an image gallery...

Note: This page is not responsive for mobile screens yet.
Here you see an image gallery based on css3d. The photos you see are from the default assets/galleries folder.
Above the canvas you can download the full gallery, if you want. On the top left within the canvas you have options for each arrangement.
Hovering (in front of camera) on image show few Exif information.
On the left in the menu you can choose an arrangement of the images. Below that you find some gloabl options.
